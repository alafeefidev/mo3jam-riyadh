# قائمة كلمات من معجم الرياض

>[>English](README_en.md)

برنامج مصنوع بواسطة لغة البرمجة "Go" للحصول على جميع الكلمات المفردة و المركبة العربية من معجم الرياض.

---
جميع الكلمات المفردة موجودة في المجلد `words_list`  
جميع الكلمات المركبة موجودة في المجلد `expressions_list`

## طريقة عمل البرنامج إذا تريد نتائج جديدة
أولا: تحميل لغة البرمجة "Go" من أي مصدر مع برنامج "Git"  

ثانيا: تطبيق الخيارات التالية
```bash
git clone https://github.com/alafeefidev/mo3jam-riyadh
cd mo3jam-riyadh
go run mo3jam.go
```

وسيتم تحديث جميع الكلمات في المجلدين
