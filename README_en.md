# Arabic Wordlist from Riyadh Dictionary

>[>ِArabic](README.md)

A script to gather all arabic words and expressions from the Riyadh dictionary written in Go.

---
All the words are found in `words_list`  
All the expressions are found in `expressions_list`

## How to Get New Data Using the Script
1. Download Go and Git
2. Use the following commands
```bash
git clone https://github.com/alafeefidev/mo3jam-riyadh
cd mo3jam-riyadh
go run mo3jam.go
```
Now all the content in both folders are updated
